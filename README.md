La tematica del proyecto es un sitio web para una agencia de viajes,donde los usuarios puedan reservar o comprar paquetes turísticos con destinos nacionales e internacionales.

ViveTrips es una página web desarrollada para una agencia de viajes, que ofrece información sobre distintos paquetes turísticos tanto nacionales como internacionales, además de promociones en paquetes ,una breve presentación sobre la empresa y una sección de contacto para comunicarse con nosotros.

El sitio esta diseñado en HTML5 y CSS3. Posee una interfaz de usuario clara y atractiva que invita a explorar la pagina y los servicios que ofrece.
